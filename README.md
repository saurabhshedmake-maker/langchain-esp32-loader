# LangChain ESP32 Loader

This project is a **LangChain-based chatbot** designed to intelligently interpret and answer technical questions from the **ESP32 datasheet**.

## 🚀 Features
- Uses LangChain for natural language understanding  
- Extracts information directly from the ESP32 datasheet  
- Built with Python and Jupyter Notebook  
- Easily extendable for other microcontrollers  

## 📂 Files Included
- `esp32ChatBot.ipynb` — Main chatbot notebook  
- `esp32_datasheet_en.pdf` — Reference datasheet  
- `requirements.txt` — Python dependencies  
- `test.py` — Basic testing script
